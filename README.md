# Grandma Left Me a Building

2D apartman yönetim simülasyonu. Casual, turn-based, cartoon tarzı.

Babaannesinden miras kalan harap bir apartmanı ayağa kaldır, absürt yaratık kiracılarla uğraş, kaynaklarını yönet ve şehrin en renkli apartmanını yarat.
